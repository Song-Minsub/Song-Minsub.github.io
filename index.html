👨‍💻 Song Minsub

안녕하세요! C# WPF 기반의 백엔드 위주로 작업하는 개발자입니다.
기능 중심의 소프트웨어 구현과 구조화에 집중하며, 필요한 경우 기본적인 화면 구성도 직접 구현합니다.

🛠️ 기술 스택

  • C#, .NET Framework (4.7.2)
  • WPF (MVVM 구조 포함)
  • MSSQL / T-SQL
  • TCP/UDP Socket 통신
  • Git (TortoiseGit 사용)

📂 프로젝트 예시

  • WPF 기반 알림 시스템 – TCP 통신 기반의 실시간 알림 기능
  • 자동 시작 유틸 – 설치 시 윈도우 시작 프로그램 등록 기능
  • DB 설정 관리 도구 – 설정 파일 기반의 DB 연결 관리기
